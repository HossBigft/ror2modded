# MultiplayerPause

Allows you to pause in multiplayer, type **pause** in chat or press **CTRL + P** to pause.

![Preview](https://raw.githubusercontent.com/Fr4nsson/MyROR2Plugins/main/plugins/MultiplayerPause/images/Preview.png)
